paper crop

画像を四角形で切り抜くツール。上下左右のエッジ形状を個別に指定できる。キャンバスは画像周囲に余白を持ち、選択は画像範囲を越えない。破れ端は自己アフィン（Hurst指数相当）な折れ線で生成する（UI変更なし）。

エッジ形状

画像上でドラッグして切り抜き範囲を指定する。余白上でドラッグしても、選択領域は画像を越えない。

使い方
1. 画像を読み込む → 2. ドラッグで範囲指定 → 3. Crop
依存: なし（Canvas API のみ）。完全オフライン対応。
保存: 状態は保存しない。
破れ生成: midpoint displacement による自己アフィン折れ線（H固定、繊維方向=上下の異方性）。
