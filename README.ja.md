# paper crop

画像を四角形で切り抜くためのシンプルな Web ツールです。上下左右それぞれのエッジを `直線`、`紙を破った風`、`手漉き紙風`、`たわみ紙風`、`半券ミシン目風` から個別に設定できます。

- Live: https://moriwaka.github.io/papercrop/
- ライセンス: MIT

## 主な機能

- 5種類のエッジ形状を上下左右で個別に切り替え
- エッジ強度の調整
- 輪郭線・ドロップシャドウの ON/OFF
- 透過確認用のチェッカーボード背景プレビュー
- 画像入力方法
  - クリックしてアップロード
  - ドラッグ&ドロップ
  - クリップボード貼り付け
- 出力
  - PNG ダウンロード
  - クリップボードへコピー
- 言語切替（日本語 / English）

クリップボードの読み書きはブラウザ対応状況に依存し、通常はセキュアコンテキスト（`https://` または `http://localhost`）が必要です。

## 使い方

1. 画像をアップロードします（クリック / ドロップ / 貼り付け）。
2. 元画像上でドラッグして切り抜き範囲を選択します。
3. エッジ形状・エッジ強度・輪郭線・シャドウを調整します。
4. `Crop` を押して結果を確認します。
5. `Download PNG` または `Copy to Clipboard` で保存します。

## ローカルで実行

ビルドは不要です。静的ファイルとしてそのまま動作します。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

## 開発メモ

- コアのマスク処理: `edge-mask.js`
- UI とイベント処理: `app.js`
- テスト依存関係のインストール:

```bash
npm install
npx playwright install
```

- 回帰テスト:

```bash
npm test
```

- ブラウザ E2E テスト:

```bash
npm run test:e2e
```

- 全テスト実行:

```bash
npm run test:all
```
